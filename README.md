# AI Knowledge Assistant

一个从 0 开始构建的 AI 知识库助手项目。

目标不是做一个只能演示的玩具，而是逐步做成一个能写进简历、能讲清楚技术细节、能部署和评测的项目。

## 项目最终形态

- 支持上传 PDF / TXT / Markdown 文档
- 自动解析文档并切分成片段
- 使用向量检索找到相关内容
- 调用大模型生成答案
- 回答时给出引用来源
- 提供简单评测：命中率、延迟、失败样例
- 使用 Docker 一键启动

## 当前阶段

第 1 阶段：创建项目、初始化 Git、连接 GitHub。

## 学习原则

每次只做一个小闭环：

1. 先知道这一步是为了解决什么问题。
2. 再动手完成一个具体产物。
3. 最后用自己的话复述它。
